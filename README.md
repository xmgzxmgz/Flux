# ⚡ Flux (MotoViz)

> 基于 React + TypeScript 的摩托车旅行可视化与路线规划平台。

## ✨ 功能

- 🗺️ 地图可视化 — 路线展示与标注
- 🌓 暗色/亮色主题切换
- 📱 响应式布局
- 🏗️ 模块化架构 — 路由、布局、页面分离

## 🛠️ 技术栈

- **前端**: React 18 + TypeScript + Vite
- **样式**: Tailwind CSS
- **路由**: React Router v6
- **构建**: Vite + pnpm

## 🚀 快速开始

```bash
pnpm install
pnpm dev      # 开发服务器
pnpm build    # 生产构建
pnpm preview  # 预览构建结果
```

## 📁 项目结构

```
src/
├── components/     # 通用组件 (导航、布局)
├── pages/          # 页面组件
├── hooks/          # 自定义 Hooks (useTheme)
├── features/       # 功能模块 (规划中)
├── router.tsx      # 路由配置
└── main.tsx        # 入口
```
