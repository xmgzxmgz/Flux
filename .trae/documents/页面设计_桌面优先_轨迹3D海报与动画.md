# 页面设计（桌面优先）：本地隐私解析 + 3D轨迹海报/动画

## 全局设计规范
### Layout
- 桌面优先：内容最大宽度 1200–1440px；左右分栏在编辑器页使用（主画布 + 右侧属性面板）。
- 布局策略：工作台/设置页以 CSS Grid 卡片栅格；编辑器/导出页以 Flex（左画布右面板）。
- 断点：≥1280（标准桌面）、≥1536（大屏）；<1024 时右侧面板折叠为抽屉。

### Meta Information
- 默认 Title 模板：`{页面名} - 轨迹3D海报/动画`。
- Description：强调“本地解析、不上传原始轨迹、快速导出海报与动画”。
- Open Graph：使用项目封面图（本地生成缩略图后在分享时可选）。

### Global Styles（Design Tokens）
- Background：#0B0F14（深色为主）/ #FFFFFF（浅色主题可选）。
- Primary：#4F46E5；Accent：#22C55E；Danger：#EF4444。
- Typography：12/14/16/20/24/32px 级联；数字指标使用等宽字体。
- Buttons：Primary / Secondary / Ghost；Hover 提升亮度 6–10%；Disabled 降低不透明度。
- 卡片：圆角 12px；阴影轻量；分割线 1px（透明度 12–16%）。

## 1) 工作台（首页）
### Page Structure
- 顶部导航 + 主内容两段：导入与项目列表。
- 主内容使用两列 Grid：左侧“导入/最近打开”，右侧“项目卡片”。

### Sections & Components
1. 顶部导航（AppBar）
   - 左：产品名 + 当前模式标签（“本地优先/离线”）。
   - 右：按钮（设置）、可选帮助入口。
2. 导入区（ImportPanel）
   - 拖拽上传/点击选择文件；支持多文件队列。
   - 格式提示（GPX/FIT/TCX）；解析进度条；错误卡片（可复制日志）。
   - 成功后显示“基础统计小卡”（距离/时长/爬升）+ “进入编辑器”。
3. 项目列表（ProjectGrid）
   - 卡片包含：封面缩略图、项目名、更新时间、快捷操作（继续编辑/导出）。
   - 搜索框 + 排序（最近更新/名称）。

### Interaction
- 导入后自动生成封面缩略图并插入第一张卡片。
- 删除项目需要二次确认，并提示将清理本地资源。

## 2) 轨迹编辑器
### Page Structure
- 左：3D 画布（CanvasStage）占 70%；右：属性面板（Inspector）占 30%。
- 顶部工具条：模板、撤销/重做、预设视角、导出入口。

### Sections & Components
1. 顶部工具条（EditorToolbar）
   - 模板下拉：海报/动画模板切换。
   - Undo/Redo。
   - 视角预设：俯视/等距/近景。
   - 右侧主按钮：去导出。
2. 3D 画布（CanvasStage）
   - 交互：旋转/平移/缩放；双击回到居中。
   - 覆盖层：帧率/渲染状态（仅在调试/设置开启时显示）。
3. 属性面板（Inspector Tabs）
   - Tab A「外观」：主题、渐变、背景、线宽、雾化/阴影开关。
   - Tab B「信息层」：标题/副标题/日期地点；数据卡片开关与布局。
   - Tab C「动画」：类型、时长、速度曲线、相机环绕参数；预览播放/暂停。

### Interaction
- 所有调整实时反映到 3D 画布；重要操作写入历史栈支持撤销。
- 动画预览占用资源时提示“降低预览质量”开关。

## 3) 导出与发布
### Page Structure
- 左侧：导出预览（静帧或动画预览）。
- 右侧：导出参数表单 + 任务队列。

### Sections & Components
1. 导出类型选择（ExportTypeSwitch）
   - 海报（PNG/JPEG）/ 动画（MP4/WebM/GIF）。
2. 参数表单（ExportForm）
   - 分辨率预设（1080p/2K/4K）与自定义；动画帧率/码率/时长。
   - 透明背景（如支持）、水印开关（默认关闭）。
3. 任务队列（ExportQueue）
   - 显示进度、剩余时间估计、取消按钮。
   - 完成后“在文件夹中打开/复制路径”。

### Interaction
- 导出时锁定关键编辑项，避免状态漂移；允许取消并回收资源。

## 4) 设置与隐私
### Page Structure
- 单列设置表单，分组卡片：隐私、存储、性能。

### Sections & Components
1. 隐私（PrivacyCard）
   - 开关：离线模式（禁用外网请求，默认开启）。
   - 声明：明确“不上传原始轨迹文件/不上传坐标点”。
2. 本地存储（StorageCard）
   - 显示占用空间估算；按钮：清理缓存/删除所有项目（危险操作确认）。
3. 性能与渲染（PerformanceCard）
   - 预览质量：低/中/高；抗锯齿开关；动画预览降级。
   - 解析线程：显示 Worker 数（自动）。
